# 🎯 Habitum — Трекер привычек

React Native приложение для отслеживания привычек с регистрацией, красивым дизайном и локальным хранилищем.

---

## 📁 Структура проекта

```
HabitTracker/
├── App.tsx                      # Корневой компонент
├── babel.config.js
├── package.json
├── tsconfig.json
└── src/
    ├── theme/
    │   └── index.ts             # Цвета, типографика, отступы
    ├── storage/
    │   └── index.ts             # AsyncStorage: User, Habit, Completion
    ├── navigation/
    │   └── index.tsx            # Stack + Bottom Tab навигация
    └── screens/
        ├── AuthScreen.tsx       # Регистрация / вход
        ├── HomeScreen.tsx       # Главный экран — привычки на сегодня
        ├── AddHabitScreen.tsx   # Создание новой привычки
        ├── StatsScreen.tsx      # Статистика и аналитика
        └── ProfileScreen.tsx    # Профиль пользователя
```

## 🚀 Установка и запуск

### Требования
- **Node.js** v18+ (рекомендуется)
- **npm** или **yarn**
- **Android Studio** (для Android эмулятора) или **Xcode** (для iOS)
- **React Native CLI**: `npm install -g react-native-cli`

### Шаг 1 — Установить зависимости

```bash
cd HabitTracker
npm install
```

### Шаг 2 — iOS (только macOS)

```bash
cd ios && pod install && cd ..
npm run ios
```

### Шаг 3 — Android

Убедитесь что запущен эмулятор или подключено устройство:

```bash
npm run android
```

### Metro bundler

Если нужно запустить отдельно:

```bash
npm start
```

## 📱 Функционал

### Экраны

| Экран | Описание |
|-------|----------|
| **Регистрация** | Выбор аватара-эмодзи, имя, email, пароль (хранится локально) |
| **Главный** | Список привычек на сегодня с возможностью отметки, прогресс, мини-календарь |
| **Добавить** | Создание привычки: название, эмодзи, категория, частота |
| **Статистика** | Тепловая карта, процент выполнения, стрики по каждой привычке |
| **Профиль** | Аватар, статистика аккаунта, выход |

### Хранение данных
- Всё хранится в **AsyncStorage** (на устройстве)
- Нет серверной части, нет облака
- Данные сохраняются между сессиями
