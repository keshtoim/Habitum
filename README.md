# Habitum — Трекер привычек

> Мобильное приложение на React Native + Expo для отслеживания ежедневных привычек. Все данные хранятся локально на устройстве.

---

## Функционал

- **Регистрация и вход** — выбор аватара-эмодзи, имя, email, пароль
- **Главный экран** — список привычек на сегодня, прогресс-кольцо, мини-календарь последних 7 дней
- **Отметка выполнения** — одно нажатие чтобы отметить привычку как выполненную
- **Стрики** — счётчик дней подряд для каждой привычки
- **Добавление привычек** — название, описание, эмодзи, категория (здоровье, разум, работа и др.), частота
- **Статистика** — тепловая карта активности за 14 дней, процент выполнения за 7 и 30 дней
- **Профиль** — информация об аккаунте, общая статистика, выход

## Запуск

### Требования
- Node.js v18+
- Приложение [Expo Go](https://expo.dev/client) на телефоне

### Установка

```bash
git clone https://github.com/ВАШ_НИК/HabitTrackerApp.git
cd HabitTrackerApp
npm install
```

### Запуск

```bash
npx expo start
```

Отсканируй QR-код в приложении Expo Go — готово.

> Телефон и компьютер должны быть в одной Wi-Fi сети. Если нет — нажми `s` в терминале для tunnel-режима.
